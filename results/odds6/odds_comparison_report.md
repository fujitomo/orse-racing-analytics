# REQI（競走経験質指数）とオッズ情報の比較分析レポート

## 概要

本分析では、レポートのH2仮説「REQI（競走経験質指数）を説明変数に加えた回帰モデルが単勝オッズモデルより高い説明力を持つ」を検証しました。

- 分析対象: 18,162頭（最低6戦以上）
- 分析期間: データセット全期間

## REQI重み情報

**訓練期間（2010-2020年）で算出された固定重み**:

- **グレード重み**: 0.587 (58.7%)
- **場所重み**: 0.367 (36.7%)
- **距離重み**: 0.047 (4.7%)

**重み算出方法**: 各要素と勝率（win_rate）の相関係数の2乗を正規化

## 1. 相関分析結果

### 1.1 REQI（競走経験質指数）と複勝率の相関

- **REQI（競走経験質指数）**: r = 0.615, R² = 0.378, p = 0.000e+00
- **最高REQI（競走経験質指数）**: r = 0.731, R² = 0.534, p = 0.000e+00

### 1.2 オッズベース予測と複勝率の相関

- **複勝オッズベース複勝率予測**: r = 0.821, R² = 0.675, p = 0.000e+00
- **単勝オッズベース勝率予測**: r = 0.806, R² = 0.650, p = 0.000e+00

## 3. 効果サイズ比較分析（Cohen's d）

### 3.1 REQIとオッズの効果サイズ比較

**目的**: REQIの効果がオッズ情報と比べてどの程度大きいかを評価

**検証方法**:
- **Cohen's d**: 2つのグループ間の効果サイズを標準化して測定
- **解釈基準**: d=0.2（小効果）、d=0.5（中効果）、d=0.8（大効果）
- **比較対象**: 
  - 高REQI群 vs 低REQI群の複勝率差
  - 人気馬群 vs 不人気馬群の複勝率差
  - REQI効果サイズ vs オッズ効果サイズ

**実際の分析結果**:
- **REQI効果**: Cohen's d = 0.963（非常に大効果）
- **オッズ効果**: Cohen's d = 1.793（非常に大効果）
- **比較結果**: オッズの方が効果が大きいが、REQIも非常に大効果で実務的に重要

### 3.2 効果サイズの解釈

| 指標 | Cohen's d | 効果サイズ | 実務的意義 |
|------|-----------|------------|------------|
| **REQI** | 0.963 | 非常に大効果 | 実務的に非常に重要な予測指標 |
| **オッズ** | 1.793 | 非常に大効果 | 最も重要な予測指標 |
| **比較** | 0.54 | REQI/オッズ比 | REQIはオッズの約54%の効果 |

**結論**: 
- 両指標とも「非常に大効果」を示し、実務的に重要な予測指標である
- オッズの方が効果が大きいが、REQIも補助的価値を持つ
- 統合利用により、より高い予測精度が期待できる

## 4. 結論

### 4.1 統計的評価

- 最も高い相関を示した予測変数: **odds_based_place_prediction** (r = 0.821)
- 最も高い予測性能を示したモデル: **統合モデル** (R² = 0.6787)

### 4.2 実務的含意

- REQI（競走経験質指数）は競馬予測において補助的な価値を持つことが確認されました
- オッズ情報との組み合わせにより、予測精度の向上が期待できます
- 両指標は相互補完的な関係にあり、統合利用が推奨されます
